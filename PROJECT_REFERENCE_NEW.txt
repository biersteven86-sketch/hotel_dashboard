PM2 Verwaltung
--------------
Status ansehen:
  pm2 status

Logs live ansehen:
  pm2 logs hotel-dashboard --lines 200 --timestamp

Start/Stop/Restart:
  pm2 stop hotel-dashboard
  pm2 start /home/steven/hotel_dashboard/server.js --name hotel-dashboard
  pm2 restart hotel-dashboard

Autostart sichern:
  pm2 save

Ibelsa-API Nutzung
------------------
Basis-URL: https://rooms.ibelsa.com
Auth: Header x-ibelsa-key (Pflicht), x-ibelsa-system-key (optional)
Format: JSON (HTTPS)

Beispiele (siehe curl-cheatsheet.md):
  curl -H "x-ibelsa-key: $IBELSA_KEY" https://rooms.ibelsa.com/api/test/ping

Node-Helper:
  import { ibelsa } from './lib/ibelsa.js';
  const data = await ibelsa('/api/hotel/information');
